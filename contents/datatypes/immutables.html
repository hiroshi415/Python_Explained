Numbers

Integer

Whole numbers which does not include decimals.

This includes negative and positive numbers

# Output will be 6

a = 2.0+4

print(int(a))

Float

One main and only difference to integer is that float is a number with decimals.

You can also specify how many digits to show by ':.x' if you format() the data

# Output will be 6.0

a = 2.0+4

print(float(a))

Complex

For example, 2 + 3i is a complex number. This way, a complex number is defined as a polynomial with real coefficients in the single indeterminate i, for which the relation i2 + 1 = 0 is imposed. Based on this definition, complex numbers can be added and multiplied, using the addition and multiplication for polynomials.

In Python, J is used to replace i

2 + 3i == 2 + 3J

Strings

String. A string is a data type used in programming, such as an integer and floating point unit, but is used to represent text rather than numbers. It is comprised of a set of characters that can also contain spaces and numbers. For example, the word "hamburger" and the phrase "I ate 3 hamburgers" are both strings.

Users cannot add strings as a number. For example, a = '10' b = '20'. print(a+b) will not equal 30.

# Output will be 1020

a = '10'

a = '20'

print(a+b)

# Output will be 33

a = 'I ate 3 hamburgers'

print(a[6]+a[6])

# Output will be 6

a = 'I ate 3 hamburgers'

print(int(a[6])+int(a[6]))

Tuples

A Tuple is a collection of Python objects separated by commas. In someways a tuple is similar to a list in terms of indexing, nested objects and repetition but a tuple is immutable unlike lists which are mutable. Note: In case your generating a tuple with a single element, make sure to add a comma after the element.

# Output will be('python', 'geeks')

tup = ('python', 'geeks')

print(tup)

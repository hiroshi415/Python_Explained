Arithmetic Operators

Operator Definition Examples
+ Add two operands or unary plus x + y
- Subtract right operand from the left or unary minus x - y
/ Divide left operand by the right one (always results into float) x / y
* Multiply two operands x * y
% Modulus - remainder of the division of left operand by the right x % y
// Floor division - division that results into whole number adjusted to the left in the number line x // y
** Exponent - left operand raised to the power of right x ** y

These operators are used for mathematical equations

Aside from the obvious ones, here are the examples below

# Output will be the remainder, thus 1 in this case

a = 21 % 4

print(a)

# Output will not include decimals, thus 5 in this case

a = 21 // 4

print(a)

# Output will be 2 to the power of 4, thus 16 in this case

a = 2 ** 4

print(a)

Identity Operators

Operator Definition Examples
is True if the operands are identical (refer to the same object) x > y
is not True if the operands are not identical (do not refer to the same object) x < y

is and is not are the identity operators in Python. They are used to check if two values (or variables) are located on the same part of the memory. Two variables that are equal does not imply that they are identical.

# Output will False

x3 = [1,2,3]

y3 = [1,2,3]

print(x3 is y3)

x3 and y3 are list. They are equal but not identical. It is because interpreter locates them separately in memory although they are equal.

Membership Operators

Operator Definition Examples
in True if value/variable is found in the sequence 5 in x
not in True if value/variable is not found in the sequence 5 not in x

in and not in are the membership operators in Python. They are used to test whether a value or variable is found in a sequence (string, list, tuple, set and dictionary). In a dictionary we can only test for presence of key, not the value.

# Output1: True: Output2: False

y = {1:'a',2:'b'}

print(1 in y)

print('a' in y)

This is case sensitive

Logical Operators

Operator Definition Examples
and True if both the operands are true x and y
or True if either of the operands is true x or y
not True if operand is false (complements the operand) not x

# Output will be {False, True, False}

x = True

y = False

print(x and y)

print(x or y)

print(not x)
